Laravel 5 JSON API Example
==========================

Laravel 5 JSON API example implementation.

Response format:

* [JSON API](http://jsonapi.org/format/)

Uses:

* [Laravel 5](https://laravel.com/docs/5.3)
* [Codeception](http://codeception.com/)

## Installation

```bash
$ composer install
$ php artisan key:generate
$ cp .env.example .env
```

Fill in `.env` file
